kodeWeave

HTML/CSS/JS and Markdown Playground For Web Designers and Developers

WHAT IS KODEWEAVE?

kodeWeave is the perfect solution for web designers and developers; it provides a custom environment (based on user selections) to test (or fiddle with) your JavaScript, HTML, CSS and Markdown code right inside your browser.

IS IT GOOD FOR ME?

kodeWeave is great for non-techies and pro-coders for fast prototyping and small customers' projects.

kodeWeave runs as an offline application for Windows, Mac, Linux and Android users. Incorporated with many of the CSS and JavaScript libraries you love.

WHY KODEWEAVE?

Key differences from traditional editors:
* Desktop exportation
* Built-in markdown editor
* Mobile-friendliness
* Easy-to-use interface
* Free for commercial and non-profit use

JQUERY

kodeWeave gives you the freedom to utilize jQuery which is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.

AJAX

kodeWeave gives you the freedom to utilize Ajax which is a client-side script that communicates to and from a server/database without the need for a postback or a complete page refresh.
